ТЕСТОВЫЙ ДОКУМЕНТ ДЛЯ АГЕНТОВ AGNO
=====================================

Этот документ предназначен для тестирования возможностей обработки текстовых файлов агентами agno.

СОДЕРЖАНИЕ:
-----------
1. Общая информация
2. Технические требования
3. Примеры использования
4. Выводы и рекомендации

1. ОБЩАЯ ИНФОРМАЦИЯ
-------------------
Файл содержит структурированный текст на русском языке с различными элементами:
- Заголовки разного уровня
- Списки (нумерованные и маркированные)
- Технические термины
- Даты и числовые данные

Дата создания: 30 июля 2025 года
Версия: 2.0
Автор: Система тестирования agno

2. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
--------------------------
Агент должен уметь:
• Извлекать ключевую информацию
• Анализировать структуру документа
• Отвечать на вопросы по содержанию
• Создавать краткие резюме
• Обрабатывать различные форматы текста

3. ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
------------------------
Возможные сценарии:
a) Анализ технической документации
b) Извлечение метаданных
c) Создание саммари
d) Поиск конкретной информации

4. ВЫВОДЫ И РЕКОМЕНДАЦИИ
-------------------------
Данный тестовый файл позволяет проверить базовую функциональность обработки текстовых документов агентами agno. Рекомендуется использовать для валидации корректности работы с русскоязычным контентом.
